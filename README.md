# Smartphone RTK Surveyor Dashboard
Real-time monitoring for RTK APP surveys in Ahmedabad

## Quick Start
1. Double-click `run.bat`
2. Browser mein: http://localhost:8050
3. RTK Fixed status dekhein!

## Features
✅ Real-time Map Tracking  
✅ RTK Status (Fixed/Float)  
✅ Satellite Count  
✅ Altitude Graph  
✅ Auto Point Collection
